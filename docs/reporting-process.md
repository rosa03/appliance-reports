# Reporting Process

## Operating process

1. Copy `templates/appliance-report-template.md` for each reporting period.
2. Rename the copied file using `YYYY-MM-DD-appliance-status-report.md`.
3. Place the report in the correct year folder under `reports/`.
4. Update the front matter fields for the reporting date, status, and forecast.
5. Update the executive summary with the latest position, progress, risks, and decisions needed.
6. Update the category scorecard statuses and headlines.
7. Record actions, decisions, and risks in the relevant sections and logs.
8. Commit changes with a meaningful message that reflects the reporting update.
9. Export the report to PDF and store it in `exports/` if a distribution copy is needed.
